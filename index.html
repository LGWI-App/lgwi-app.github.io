About the Project

LGWI is a non-profit organization dedicated to providing clean water to communities in Ecuador. To promote fairness and sustainability, this project aims to create a mobile app that accurately measures individual household water usage.

Previously, every household contributed a flat monthly fee, which placed an unequal burden on smaller families who used less water compared to larger ones. Our app uses Optical Character Recognition (OCR) technology to automatically read water meter levels, ensuring accurate tracking and billing based on actual usage. This digital solution replaces the current handwritten system, making the process more transparent, efficient, and equitable for all community members.

Key Features

  • Feature 1: Optical Character Recognition: This feature lets the user scan water meters, and the OCR technology automatically reads and extracts the meter value.
  • Feature 2: Meter Page: This page allows the user to review all meters and see which ones have already been scanned, making it easy to identify which meters still need to be read.
  • Feature 3: Meter Cost Tracking: This feature enables the user to view and track water usage and costs for each meter, helping ensure accurate billing and proper charge distribution to the homes.
